thepalewitch

A full-stack PERN website where users can view updates on upcoming Steam games created by thepalewitch.

Live Site: https://thepalewitch.com

- Tech Stack -

Frontend

React

Vite

Backend

Node.js

Express.js

Database / Hosting

PostgreSQL

Vercel (Frontend Deployment)

Render (Backend & DB Deployment)

- Features -

User authentication (JWT + hashed passwords)

Create, edit, and delete comments

Like system

Responsive UI

REST-based Express API

Secure PostgreSQL queries

- Installation & Setup -
  Frontend
  cd Frontend
  npm install
  npm run dev

Backend
cd Backend
npm install
nodemon server.js
